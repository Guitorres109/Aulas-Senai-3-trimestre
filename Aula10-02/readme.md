# Principais tipos de dados Javascript

### String: (Texto):

- Sempre é escrita entre aspas (" " ou ' ')

### Number: (numero):

- No JavaScript, número podem ser inteiros ou decimais

### Boolean: (verdadeiro ou falso):

- Um boolean pode ter apenas dois valores: true e false

### Object: (objeto):

- Um objeto agrupa várias informações em uma só variavel

### Anotações:

- Usar 'typeof' para descobrir tipo de um valor ou variavel
- Template String: ```console.log(`Eu moro em ${cidade}.`)```

### Camel Case:
- Primeira palavra comaça com letras minusculas
- Palavras seguintes comaçam com letras maiuscula
- ```let idadeUsuario = undefined```

### Pascal Case:
- Todas as palavras começam com a letra maiuscula
- Usada por classes, construtores e componentes
- ```let IdadeUsuario = undefined```

### Snake Case:

- Palavras Separadas por underscore "_"
- Tudo Minusculo
- Muito comum em python, banco de dados e APIs
- ```let nome_completo_casado```

### Kebab Case:
- Palavras separadas por hífen "-"
- Não funciona como nomes de variaveis em JavaScript
- ```<div class="menu-principal"></div>```
